## Manthan Tax Consultancy — Website

A multi-page business website built for Manthan Tax Consultancy, designed to showcase their tax and financial advisory services and provide an easy way for clients to get in touch.

__Overview__

This is a static front-end website using a custom "Midnight Gold" theme — a dark, premium color palette paired with gold accents to convey trust and professionalism, fitting for a tax consultancy brand.

__Tech Stack__


- HTML5 — page structure and content
- CSS3 — custom styling, Midnight Gold theme, responsive layout
- JavaScript (Vanilla) — interactivity and form handling
- Email integration — contact form connected to email delivery for client inquiries


__Features__

-Multi-page layout (e.g., Home, Services, About, Contact)
-Responsive design for desktop and mobile
-Custom Midnight Gold visual theme
-Functional contact form with email integration
-Clean navigation for easy access to services and information
